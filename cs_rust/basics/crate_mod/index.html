CrateгҖҒModгҖҒеҸҜи§ҒжҖ§гҖҒж–Үд»¶з»“жһ„

еҸ‘иЎЁдәҺ 2019-03-21 жӣҙж–°дәҺ 2019-02-17 еҲҶзұ»дәҺ Rust
CrateгҖҒModгҖҒеҸҜи§ҒжҖ§гҖҒж–Үд»¶з»“жһ„

ж–Үд»¶з»“жһ„и§„еҲҷ

жӢҶеҲҶж–Үд»¶

RustдёҖиҲ¬еә“дёӯж–Үд»¶еҗҚ/ж–Үд»¶еӨ№йғҪиЎЁзӨәmod пјҲжөӢиҜ•ж–Үд»¶и§„еҲҷжҜ”иҫғзү№ж®Ҡпјү

  • modfooеңЁе…¶зҲ¶modbarдёӯеЈ°жҳҺ
  • еҰӮжһңmodfooжІЎжңүеӯҗmodпјҢе°Ҷе…¶е®һзҺ°ж”ҫеңЁfoo.rsж–Үд»¶дёӯ
  • иӢҘmodfooжңүеӯҗmodпјҢеҲӣе»әж–Үд»¶еӨ№fooпјҢе°Ҷе…¶е®һзҺ°ж”ҫеңЁ foo/mod.rsдёӯ

д»ҘдёҠжҳҜж–Үд»¶жӢҶеҲҶи§„еҲҷпјҢд№ҹеҸҜд»ҘдёҚжӢҶеҲҶж–Үд»¶

еә“Crate

еә“crateдёӯlib.rsзӣёеҪ“дәҺиҜҘcrateйЎ¶еұӮmodпјҲж №modпјү

  • жүҖжңүзҡ„modзӣҙжҺҘжҲ–й—ҙжҺҘпјҲзҘ–е…ҲmodпјүеЈ°жҳҺдәҺжӯӨпјҢеҗҰеҲҷдёҚиғҪиҜҶеҲ«
  • д»Һеј•з”Ёеә“crateзҡ„еӨ–йғЁcrateи§’еәҰжқҘзңӢпјҢе…¶еҗҚз§°е’Ңеә“crateеҗҢеҗҚ extern crate crate_name;зҡ„еҗҢж—¶е°ұuse crate_name;пјҢ жӯӨж—¶еҸҜе°Ҷеј•з”Ёе…¶зҡ„modи§Ҷдёәж №modзҡ„зҲ¶mod

еә“гҖҒдәҢиҝӣеҲ¶Crate

crateдёӯеҸҜд»ҘеҗҢж—¶жңүlib.rsе’Ңmain.rsпјҢжӯӨж—¶еә“crateе’ҢдәҢиҝӣеҲ¶ crateеә”иҜҘзңӢдҪңзӣёдә’зӢ¬з«Ӣ

  • еңЁдёӨеӨ„йғҪдҪҝз”Ёmodе…ій”®еӯ—еЈ°жҳҺе®ҡд№үmodпјҲдёҚиғҪеңЁmain.rs дёӯдҪҝз”ЁuseеЈ°жҳҺдҪҝз”Ёmodпјү

  • еңЁmain.rsдёӯдҪҝз”Ёextern crate crate_nameеј•е…Ҙ вҖңеӨ–йғЁвҖқеә“crate

еҸҜи§ҒжҖ§и§„еҲҷ

Modй»ҳи®Өз§Ғжңү

  • й»ҳи®Өд»…crateеҶ…йғЁеҸҜи§Ғ
    • зҲ¶modеӨ„зӣҙжҺҘеҸҜз”Ё
    • е…„ејҹmodгҖҒеӯҗmodеҸҜд»ҘйҖҡиҝҮвҖңеӣһжәҜвҖңеЈ°жҳҺдҪҝз”Ё
  • pubеЈ°жҳҺдёәе…¬з”ЁеҗҺпјҢеҜ№еӨ–йғЁcrateд№ҹеҸҜи§Ғ

Fnй»ҳи®Өз§Ғжңү

  • й»ҳи®Өд»…modвҖңеҶ…йғЁвҖқеҸҜи§ҒпјҲеҢ…жӢ¬еҗҺд»Јmodпјү
    • еҪ“еүҚmodеҶ…зӣҙжҺҘеҸҜз”Ё
    • еӯҗmodеҸҜд»ҘйҖҡиҝҮвҖңеӣһжәҜвҖқеЈ°жҳҺеҸҜз”Ё
  • pubеЈ°жҳҺдёәе…¬з”ЁеҗҺпјҢеҜ№еӨ–йғЁmodд№ҹеҸҜи§Ғ

иҜҙжҳҺ

  • йЎ№пјҲmodгҖҒfnпјүзҡ„еЈ°жҳҺдҪҝз”Ёи·Ҝеҫ„йғҪжҳҜзӣёеҜ№дәҺеҪ“еүҚйЎ№пјҢеҚій»ҳи®Өи°ғз”Ё е…¶еҗҺд»ЈйЎ№пјҲmodгҖҒfnпјүпјҢйҖҡиҝҮд»ҘдёӢвҖңеӣһжәҜвҖқж–№ејҸи°ғз”ЁйқһзӣҙжҺҘеҗҺд»ЈйЎ№
    • superзӣҙжҺҘзҲ¶modи·Ҝеҫ„иө·е§Ӣпјҡsuper::child_mod
    • ::ж №modиө·е§Ӣпјҡ::child_mod
  • fnе’Ңmodзҡ„еҸҜи§Ғи§„еҲҷзӣёдјјзҡ„пјҢеҸӘжҳҜжіЁж„ҸпјҡfnжҳҜеҗҰеҸҜи§ҒеҸӘдёҺmodжңүе…і пјҢmodжҳҜеҗҰеҸҜи§ҒеҸӘжңүcrateжңүе…ігҖӮд»ҺиҝҷдёӘж„Ҹд№үдёҠиҜҙпјҢcrateдёҚиғҪ зңӢдҪңжҳҜвҖңеӨ§еҸ·вҖңзҡ„mod

зӣёе…іе…ій”®еӯ—

еҘҪеғҸйғҪжҳҜеҚ•дёҖз”ЁйҖ”пјҲж„Ҹд№үпјүпјҢзҪ•и§Ғ

  • externпјҡеј•е…ҘеӨ–йғЁcrateпјҲеҗҢж—¶еҢ…еҗ«use crate_name;пјү
  • crateпјҡж Үи®°еӨ–йғЁcrate
  • mod/fnпјҡеЈ°жҳҺе®ҡд№үпјҲжіЁеҶҢпјүmod/fnпјҲеҗҢcrateеҶ…д»…дёҖж¬Ў пјҢдҪҚдәҺе…¶зҲ¶modеӨ„пјү
  • useпјҡеЈ°жҳҺдҪҝз”ЁйЎ№пјҲmodгҖҒfnпјүпјҢз”ЁдәҺзј©з•Ҙд»Јз Ғ
