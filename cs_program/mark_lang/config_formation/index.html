特殊符号

  • ---：表示文档开始
  • ...：文档结束
    • 二者配合在文件中记录多个yaml配置项
  • !!：强制类型转换
  • >：折叠换行符为空格
  • |：保留换行符
  • &：锚点
    • 不能独立定义，即非列表、映射值
  • *：锚点引用
    • 可以多次引用
    • 被引用值可能会之后被覆盖
  • <<：合并内容
    • 主要配合锚点使用
    • 相当于unlist解构

    • package：org.yaml.snakeyaml.Yaml
  • Python
    • package：PyYaml
